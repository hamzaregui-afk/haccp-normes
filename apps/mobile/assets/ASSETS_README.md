# Mobile App Assets — Required Before Store Submission

These files must be created BEFORE building for App Store or Play Store.

## Required Files

| File | Size | Description |
|------|------|-------------|
| `icon.png` | 1024×1024 px, PNG | App icon (iOS + Android) |
| `splash.png` | 1080×1920 px, PNG | Splash screen |
| `adaptive-icon.png` | 1024×1024 px, PNG | Android adaptive icon foreground (transparent bg) |

## Brand Colors
- Background: #1A3D2B (dark green)
- Accent: #B5833A (gold)
- Text on dark: #FFFFFF

## Design Guidelines
- icon.png: NORMES HACCP logo on #1A3D2B background, no rounded corners (Expo adds them)
- adaptive-icon.png: Just the logo symbol, transparent background (Android adds the shape)
- splash.png: Centered logo on #1A3D2B background with tagline

## Tools to Create
- Figma / Canva / Adobe Illustrator
- Export as PNG at the required sizes
- Do NOT use transparency in icon.png (iOS requires opaque icon)
